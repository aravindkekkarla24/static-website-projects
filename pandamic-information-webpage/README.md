# Pandemic Information Webpage

This is a static informational website created to spread awareness about a pandemic. The website presents general information, symptoms, and related awareness content in a simple and easy-to-understand layout.

This project was built as part of my learning journey using HTML and CSS.

## Features
- Informative pandemic awareness content  
- Simple and clean UI design  
- Easy navigation layout  
- Fully static webpage  

## Technologies Used
- HTML  
- CSS  

## What I Learned
- Structuring content using HTML  
- Designing layouts using CSS  
- Creating clean and readable UI  
- Building static informational websites
